LDAP Integration
(CSEE with iPlanet's LDAP5.0 or LDAP5.1)
Author: Krishna Gorthi
Version: 2.1

This document provides you with the basic steps to integrate CSEE with iPlanet's LDAP5.0/5.1. This document assumes that you have upgraded/installed the CSEE successfully.

On successfully integrating CSEE with LDAP :
You can do :

  • Adding a new user
  • Adding a new ACL to the user
  • Deleting a user
  • Searching for Users
    NOTE: Users should have administative privilages on LDAP to do the first three operations.
                 You might want to consult your LDAP admin for configuring your users and groups.

you cannot do :

  • Removing ACLs from a user
  • Adding a new ACL

  4. Get the latest version of LDAP Integration tool and import it through the project option in the Content Server Explorer.

    5.           NOTE: The tool is written with JSPs so, you should configure your appserver to run JSPs to use this tool..

    •  Invoke the following url from the browser after filling in the host, port and appserver root values.
                  http://host:port/<appserver root>/ContentServer?pagename=LDAP/Integrate
                  Example: http://localhost:80/servlet/ContentServer?pagename=LDAP/Integrate
    • Follow the steps given on each screen of the LDAP Integration tool.
    • Strictly follow the notes and cautions on the screen while using this tool inorder to integrate the CSEE & LDAP hassle free.

  5. You have now successfully integrated the LDAP with CSEE.
